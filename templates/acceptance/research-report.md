# Acceptance Criteria: Research Report

Copy this file to `.quiver/acceptance.md` in your workspace to define what a
complete, high-quality research report looks like. The maker-checker will
verify each criterion before the agent writes the final deliverable.

## Structure
- [ ] Report has a clear title and date
- [ ] Report has an executive summary (2-3 paragraphs)
- [ ] Report has a methodology section explaining sources used
- [ ] Report has numbered sections with descriptive headings
- [ ] Report has a conclusion with actionable recommendations

## Content Quality
- [ ] Every factual claim has an inline citation with source URL
- [ ] No claim is stated without a verifiable source
- [ ] Conflicting evidence is acknowledged, not hidden
- [ ] Report addresses the original research question directly
- [ ] Report includes limitations and caveats of the analysis

## Sources
- [ ] At least 5 distinct sources are cited
- [ ] Sources include primary sources where available (filings, official documents, original research)
- [ ] Sources are recent (within 2 years unless historical context is needed)
- [ ] Source URLs are valid and accessible

## Format
- [ ] Report is written in clear, professional prose
- [ ] No placeholder text (TODO, FIXME, lorem ipsum)
- [ ] No unexplained jargon or acronyms
- [ ] Report length is appropriate for the topic (minimum 1000 words)
